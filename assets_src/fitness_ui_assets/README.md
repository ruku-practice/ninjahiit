# Fitness UI Assets

生成済みの縦型スマホUI案から、実装用の参照素材として切り出したPNG素材です。

## フォルダ構成

- `variant_01` 〜 `variant_06`
  - `reference/full_mockup.png`：全体UI
  - `background/`：背景・ヘッダー・キャラ周辺の参照用切り出し
  - `menu_parts/`：右側のワークアウトメニュー、カード、ステータスウィジェット
  - `footer/`：フッターナビ全体と各タブの切り出し

## 注意

元画像はレイヤー付きデザインデータではなく、1枚絵として生成されています。
そのため、キャラクターやUIの裏に隠れている背景、カード単体の完全な透明素材は復元できません。

実装では、このZIP内のPNGを「仮素材・デザイン参照」として使い、
Flutter / Godot / Web などのUIコンポーネントとして再構築するのがおすすめです。
